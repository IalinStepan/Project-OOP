## Описание проекта
Проект представляет собой программную модель теплообменника и расходомера для систем отопления и водоснабжения. 
Реализованы два класса с валидацией данных, расчетами и тестированием.

## Классы

### HeatExchanger (Теплообменник)
Класс для моделирования теплообменника.

**Атрибуты:**
- `id` - уникальный идентификатор
- `model` - модель теплообменника
- `area` - площадь теплообмена (м²)
- `max_temp` - максимальная температура (°C)
- `max_pressure` - максимальное давление (бар)
- `medium` - рабочая среда

**Методы:**
- `calculate_heat_flow(flow_rate, delta_t)` - расчет теплового потока
- `heat_flow` - последний рассчитанный тепловой поток
- `delta_t` - последняя разница температур
- `power_per_area` - мощность на единицу площади
- `theoretical_max_power` - теоретическая максимальная мощность

### FlowMeter (Расходомер)
Класс для моделирования расходомера.

**Атрибуты:**
- `id` - уникальный идентификатор
- `model` - модель расходомера
- `type` - тип расходомера
- `diameter` - диаметр (мм)
- `flow_range` - диапазон измерений (кг/с)
- `error` - относительная погрешность (0-1)

**Методы:**
- `get_flow()` - получить случайное значение расхода
- `last_flow` - последнее измеренное значение
- `max_error` - максимальная абсолютная погрешность
- `error_percent` - погрешность в процентах
- `flow_range_lmin` - диапазон в л/мин
